# 📝 DevNotes

Aplicação web simples para criação e gerenciamento de notas diretamente no navegador.

Link: 

## 🚀 Funcionalidades

* ➕ Adicionar novas notas
* ✏️ Editar notas em tempo real
* 📌 Fixar notas importantes
* 📋 Duplicar notas
* ❌ Excluir notas
* 🔍 Buscar notas por conteúdo
* 📤 Exportar notas em formato CSV

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage

## 💾 Armazenamento

As notas são armazenadas no **LocalStorage do navegador**, permitindo que os dados sejam mantidos mesmo após recarregar a página.



## 📚 Aprendizados

Este projeto foi desenvolvido durante um curso de formação Front-End, com foco em:

* Manipulação do DOM
* Eventos em JavaScript
* Estruturação de código
* Persistência de dados no navegador
